# 0.3.0

# 功能

# @logicflow/core

  • 支持 mini-map 能力
  • 支持框选能力
  • 支持动态修改画布的编辑类配置
  • 支持设置节点的锚点数量和位置
  • 支持设置连线文本背景的 hover 样式 - hoverBackground
  • 支持设置连线的虚线样式 - strokeDashArray
  • 新增节点和画布的拖拽相关事件
  • 优化自定义元素的开发体验

# @logicflow/extension

  • 新增 mini-map 组件
  • 新增框选组件
  • 新增 Turbo 数据转换器

# API

# 新增

  • updateEditConfig - 更新画布的编辑类配置
  • getEditConfig - 获取画布的编辑类配置
  • getSelectElements - 获取选中的所有元素
  • clearSelectElements - 取消所有元素的选中状态
  • updateText - 更新元素的文本内容
  • setAttributes - 设置元素 Model 中的数据
  • anchorsOffset - 元素锚点相对于元素中心的偏移量
  • sourceRules - 连线时，元素作为起始节点的校验规则
  • targetRules - 连线时，元素作为目标节点的校验规则

# 变更

  • focusOn - 删除 type 参数，LogicFlow 可以直接通过 id 区分元素类型

# 兼容性

无

# Bug

  • 修复当 textEdit 为 false 时，不能触发双击事件的问题
  • 修复节点被选中时，画布容器出现边框的问题
  • 修复ctrl+v不能触发 beforeClone guard 的问题
  • 修复 extension 重复加载的问题
  • 修复 vue2 的兼容性问题

# 文档

  • 规范节点和连线的自定义方式，点击查看详情
  • 修改若干错误
